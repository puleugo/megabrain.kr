# 인제대학교 통학 버스 예약 시스템 개선
<b>2024 학술제</b><br/>
행사 기간: 2024.11.27.17:00 ~ 2024.11.27.20:00

### 프로젝트 소개
기존 인제대학교 통합 버스 예약 시스템의 불편함을 개선하기 위해 개발하였습니다.
사용자 인터뷰를 통해 기존 시스템의 UI/UX에 여러 문제점이 있다는 것을 확인하였고,
학우들의 피드백을 반영하여 더 직관적이고 효율적인 버스 예약 시스템을 구축하는 것을 목표로 하고 있으며,
이를 통해 학생들의 불편함을 해소하고 사용자 경험을 크게 향상시키고자 합니다.

- [프로젝트 주소](https://inje-bus.vercel.app/Login)
- [Github](https://github.com/inje-megabrain/irBs)

### 참여자
- 서상혁    
- 문보경
- 임성빈
- 제영선
- 서영준

### 주요 기능
- <b>직관적인 정보 배치</b><br/>
    중요한 정보를 메인 페이지에 배치하여, 사용자가 필요한 내용을 빠르게 확인할 수 있도록 접근성을 높였습니다.
- <b>간소화된 기능 제공</b><br/>
    불필요한 기능을 제거하고, 핵심 기능에 집중하여 사용자의 혼란을 최소화 하였습니다.
- <b>모바일 최적화 UI/UX</b><br/>
    모바일 환경에서의 사용성을 고려한 UI/UX 설계로, 터치 기반의 직관적인 메뉴 구성을 구현하여 사용자의 편의를 극대화하였습니다.
- <b>실 사용으로 긍정적 경험 제공</b><br/>
    API를 활용하여 사용자가 직접 사용 가능하도록 설계하였습니다.
